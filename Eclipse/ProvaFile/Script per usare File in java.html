  • [JAVA] Scrivere, leggere e modificare i file

    Pubblicato il marzo 16th, 2009 admin 3 commenti

    Ecco una serie di buoni pezzi di codice per riuscire al meglio nella manipolazione dei file in Java. Potrete leggerli, scriverli e modificarli in tutta facilità.

    Per aprire un file Java occorre scrivere:

    File f=new File("nomefile.txt");

    Script per leggere e mostrare a video il contenuto di un file:

    File f=new File("prova.txt");
FileInputStream fis=new FileInputStream(f);
InputStreamReader isr=new InputStreamReader(fis);
BufferedReader br=new BufferedReader(isr);
String linea=br.readLine();
while(linea!=null) {
       System.out.println(linea);
       linea=br.readLine();
}

    Questo script invece realizza la scrittura di un testo nel file txt

    File f=new File("prova.txt");
FileOutputStream fos=new FileOutputStream(f);
PrintStream ps=new PrintStream(fos);
ps.println("Hello Word!");

    Se invece di creare (o sovrascrivere in caso di esistenza già del file) volessimo scrivere in coda a un file già esistente bisognerà aprire il file con la seguente istruzione:

    FileOutputStream fos=new FileOutputStream(f,true);

    Ed ecco infine l’ultimo script per leggere tutti i nomi dei file contenuti in una particolare cartella:

    File f=new File("c:\\");
if (f.isDirectory()) {
    String files[]=f.list();
    for(int i=0;i<files.length;i++)
         System.out.println(files[i].getName());
}
